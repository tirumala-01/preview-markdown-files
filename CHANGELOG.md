# Changelog

All notable changes to Auto Preview Markdown will be documented in this file.

## [1.0.0] - 2025-01-09

### Added

- Auto-open preview when opening markdown/markup files
- Auto-close preview when source file is closed
- Preview-only mode for distraction-free reading
- Smart git diff detection to prevent unwanted previews
- Support for Markdown, AsciiDoc, and reStructuredText
- Configurable settings for all behaviors
- Keyboard shortcuts for toggling features
- Focus preservation option to keep cursor in source editor
